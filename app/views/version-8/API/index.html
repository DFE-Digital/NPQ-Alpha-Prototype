Get started

Apply allows you to integrate your software with ours and test it. It also lets you process dummy applications to test as a training provider.

Apply - what’s it about

Apply replaces the online UCAS application system for postgraduate teacher training

All vendors of student record systems and some training providers must make changes to integrate with Apply.

How is it used

When Apply receives an application, it can then be seen by others

Providers can then use it to:

  • retrieve applications
  • make a candidate an offer
  • Confirm a candidate’s suitability
  • reject an application

How do I connect to Apply

Any request needs validating by an authentication token.

A token is created by a provider. It grants access to applications made by that provider.

Get a token by emailing becomingateacher@digital.education.gov.

To activate a token, see the Apply reference.
